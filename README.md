# NeverGiveUp
# 项目起源
大二下的Qt实训课程需要使用Qt写一个作品。
# 项目环境
本项目全程在Felgo下开发。
# 项目简介
本项目是模仿手游《永不言弃》，大致的游戏内容相似，但是加入了我们自己的一些创新。
* 增加了死亡情况。<br>
  在《永不言弃》中，不会出现掉落到跳台下的情况发生，本项目在关卡中多处设置空位置，若玩家未跳过则会掉落到游戏场景边界，导致死亡。
* 增加了人物的粒子效果<br>
  在人物的背后会有小星星粒子不断的出现，使人物看起来移动更加生动。
